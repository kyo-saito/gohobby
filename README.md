# GoHobby - 目標達成とリワードシステム

働く人のストレス軽減と幸福度上昇を目的とした、目標達成とリワードシステムのアプリです。

## コンセプト

現代人のストレスの上昇の原因が、ストレスを受けるタイミングと報酬を受けるタイミングの間にラグがあるために、脳の報酬系がうまく働かず、ストレスだけが蓄積されているという仮説に基づいています。

自分で立てた目標を達成してリワードを得ることで、日々のストレスを報酬のために行っていると思えるようにし、報酬系を機能させてストレスの蓄積を防ぎます。

## 主要機能

1. **目標を登録する機能** - ユーザーが仕事の目標を設定・登録
2. **リワードを登録する機能** - 目標達成時に受け取るリワードを事前に設定
3. **目標が達成できたことを報告する機能** - 目標達成を記録・報告
4. **リワードを受け取る機能** - 達成報告後にリワードを受け取る

## 技術スタック

- **フロントエンド**: React 18 + Vite
- **スタイリング**: Tailwind CSS
- **状態管理**: React Context API + useReducer
- **データ永続化**: localStorage

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ファイル変更時に自動でバージョンを更新（別ターミナルで実行）
npm run watch:version

# ビルド
npm run build

# プレビュー
npm run preview
```

## バージョン管理

- **自動バージョン更新**: ファイルを変更して保存すると、自動的にバージョンが更新されます
  - `npm run watch:version` を別ターミナルで実行すると、ファイル変更を監視して自動更新
  - Gitコミット時にも自動的にバージョンが更新されます（pre-commitフック）
- **手動バージョン更新**: `npm run version:bump` で手動更新も可能

## 使用方法

1. ホーム画面から「目標を登録」をクリック
2. 目標名と説明を入力して登録
3. リワードを設定（目標達成時のご褒美）
4. 目標を達成したら「目標を達成した！」ボタンをクリック
5. リワードを受け取る

## デモシナリオ

1. ホーム画面を表示（初期状態）
2. 目標を登録（例：「今週のプレゼン資料を完成させる」）
3. リワードを登録（例：「好きなレストランでディナー」）
4. 目標達成を報告
5. リワードを受け取る（達成感の確認）
