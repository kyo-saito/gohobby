# アイコン画像の配置方法

## 手順

1. 添付されたアイコン画像（拳＋星付きギフトのアイコン）を `icon.png` という名前で保存してください
2. このファイル（`public/README_ICON.md`）と同じ `public` フォルダに配置してください
3. ファイルパス: `public/icon.png`

## 推奨仕様

- **サイズ**: 512x512px または 1024x1024px
- **形式**: PNG（透明背景推奨）
- **ファイル名**: `icon.png`（小文字）

## 確認方法

画像を配置後、ブラウザで以下を確認してください：
- ブラウザタブのfaviconに表示される
- ヘッダー左上のロゴに表示される

画像が読み込めない場合は、自動的に絵文字アイコン（🎯）が表示されます。
